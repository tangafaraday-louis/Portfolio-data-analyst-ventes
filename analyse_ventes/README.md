# 📊 Analyse des performances des ventes

## 📌 Objectif

Analyser les données de vente afin d’identifier les leviers d’amélioration de la rentabilité et de la performance commerciale.

---

## 🛠️ Outils utilisés

- Excel (analyse, TCD, nettoyage)
- Power BI (tableau de bord interactif)
- SQL (extraction et transformation)

---

## 📊 Indicateurs clés

- Chiffre d’affaires
- Profit
- Marge bénéficiaire
- Nombre de commandes

## 📈 Insights clés

- Forte augmentation des ventes en fin d’année (saisonnalité)
- Les remises élevées impactent fortement la rentabilité
- Certaines régions génèrent du chiffre d’affaires mais peu de profit
- Les produits les plus vendus ne sont pas toujours les plus rentables

---

## 📌 Recommandations

- Réduire les remises élevées (>20%)
- Prioriser les produits à forte marge
- Adapter la stratégie commerciale par région
- Exploiter la saisonnalité pour maximiser les ventes

---

## 🎯 Conclusion

Ce projet montre l’importance d’une analyse orientée rentabilité pour améliorer la performance globale d’une entreprise.
